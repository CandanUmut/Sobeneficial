# Code of Conduct | Davranış Kuralları

[English](#english) | [Türkçe](#turkce)

## English
SoBeneficial exists to deliver benefit, not harm. Everyone interacting in GitHub issues/PRs/discussions or community channels is expected to help maintain a safe, inclusive, and mission-aligned environment.

### Expected behavior
- Be respectful and constructive; critique ideas, not people.
- Welcome newcomers and share knowledge generously.
- Protect privacy; avoid sharing personal data without consent.
- Use clear, non-clickbait titles and evidence-based discussion.
- Elevate safety: if something feels off, call it out calmly.

### Unacceptable behavior
- Harassment, threats, stalking, or targeted bullying
- Hate speech, discrimination, or slurs
- Doxxing or revealing personal information without consent
- Unwanted sexual content or advances
- Persistent disruption, spam, or bad-faith contributions

### Scope
These rules apply to all project spaces: GitHub repositories (issues, PRs, discussions), community forums/chats, events, and any platform representing the project.

### Reporting
- For conduct issues: reach out to maintainers via GitHub Discussions or the contact email placeholder `safety@sobeneficial.org` (replace when available). Include links, context, and screenshots when possible.
- For serious security vulnerabilities or risk of harm: **do not** open a public issue. Use GitHub Security Advisories for responsible disclosure as outlined in [SECURITY.md](./SECURITY.md).
- If you are unsure where to report, start privately; we will route it appropriately.

### Enforcement
- Maintainers may warn, mute, block, or remove participants violating these rules.
- Severe violations (harassment, doxxing, safety threats) may result in immediate removal and coordinated response with hosting providers.
- Appeals can be requested via the same reporting channel; we aim for transparent, accountable decisions.

### Commitment to safety-first culture
SoBeneficial prioritizes community health over growth. Moderation decisions will err on the side of protecting vulnerable members, especially in sensitive communities and Donated Sessions contexts.

## Türkçe
SoBeneficial zararı değil faydayı amaçlar. GitHub issue/PR/tartışmaları veya topluluk kanallarında bulunan herkesin güvenli, kapsayıcı ve misyona uygun bir ortamı koruması beklenir.

### Beklenen davranışlar
- Saygılı ve yapıcı olun; insanları değil, fikirleri eleştirin.
- Yeni gelenleri karşılayın, bilginizi paylaşın.
- Gizliliğe özen gösterin; izinsiz kişisel veri paylaşmayın.
- Net ve kanıta dayalı iletişim kurun; clickbait başlıklardan kaçının.
- Güvenliği yükseltin: yanlış giden bir şey görürseniz sakin şekilde belirtin.

### Kabul edilemez davranışlar
- Taciz, tehdit, takip veya hedef gösterme
- Nefret söylemi, ayrımcılık, hakaret
- İzinsiz kişisel bilgi ifşası (doxxing)
- İstenmeyen cinsel içerik veya yaklaşım
- Sürekli rahatsızlık verme, spam, kötü niyetli katkılar

### Kapsam
Kurallar; GitHub (issue, PR, tartışmalar), topluluk forum/sohbetleri, etkinlikler ve projeyi temsil eden tüm platformlar için geçerlidir.

### Bildirim
- Davranış sorunları için: GitHub Discussions üzerinden veya `safety@sobeneficial.org` (yerine gerçek iletişim eklenince güncellenecek) adresine özel mesaj atın. Bağlam, bağlantı ve ekran görüntüsü ekleyin.
- Ciddi güvenlik açıkları veya zarar riski için: **public issue açmayın.** [SECURITY.md](./SECURITY.md) içindeki GitHub Security Advisories sürecini kullanın.
- Emin değilseniz özelden yazın; uygun kanala yönlendiririz.

### Yaptırım
- Kuralları ihlal edenlere uyarı, sessize alma, engelleme veya uzaklaştırma uygulanabilir.
- Ağır ihlallerde (taciz, doxxing, güvenlik tehdidi) anında kaldırma ve gerekirse barındırma sağlayıcılarıyla koordinasyon yapılır.
- İtirazlar aynı iletişim kanalı üzerinden iletilebilir; kararları şeffaf tutmaya çalışacağız.

### Güvenlik-öncelikli kültüre bağlılık
SoBeneficial büyümeden önce topluluk sağlığını önemser. Moderasyon kararları, özellikle hassas topluluklar ve Bağış Seansları bağlamında, savunmasız üyeleri koruma lehine verilecektir.
